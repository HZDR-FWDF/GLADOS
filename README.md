# ddrf - Dresden Reconstruction Framework

## Build instructions

### Dependencies

* Boost.System 1.54.0
* Boost.Filesystem 1.54.0
* Boost.Log 1.54.0
* Boost.Date_Time 1.54.0
* CUDA 7.5
* a CUDA 7.5 compatible C++ compiler with C++11 support (e.g. gcc 4.9)

### Building

Build the project in release mode.
